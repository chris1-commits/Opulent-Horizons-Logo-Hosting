# Icons

This directory contains icons and small graphical elements.

## Usage

Images in this directory can be accessed via GitHub's raw content URL:
```
https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/icons/[filename]
```

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `menu-icon.svg`
- `search-icon.png`
- `user-avatar-default.png`
