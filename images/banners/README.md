# Banners

This directory contains banner images for websites, emails, and marketing.

## Usage

Images in this directory can be accessed via GitHub's raw content URL:
```
https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/banners/[filename]
```

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `homepage-banner.jpg`
- `email-header.png`
- `promotion-banner-1200x400.png`
