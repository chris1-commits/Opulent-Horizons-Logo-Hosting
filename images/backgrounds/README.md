# Backgrounds

This directory contains background images and patterns.

## Usage

Images in this directory can be accessed via GitHub's raw content URL:
```
https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/backgrounds/[filename]
```

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `hero-background.jpg`
- `texture-pattern.png`
- `gradient-blue.svg`
