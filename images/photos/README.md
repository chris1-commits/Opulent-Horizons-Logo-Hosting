# Photos

This directory contains general photos and images.

## Usage

Images in this directory can be accessed via GitHub's raw content URL:
```
https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/photos/[filename]
```

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `team-photo-2024.jpg`
- `office-exterior.jpg`
- `product-showcase.png`
