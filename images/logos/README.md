# Logos

This directory contains company and product logos.

## Usage

Images in this directory can be accessed via GitHub's raw content URL:
```
https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/logos/[filename]
```

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `company-logo.png`
- `product-logo-dark.svg`
- `brand-icon.png`
