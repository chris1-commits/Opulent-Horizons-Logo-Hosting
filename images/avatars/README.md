# Avatars

This directory contains user avatars and profile pictures.

## Usage

Images in this directory can be accessed via GitHub's raw content URL:
```
https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/avatars/[filename]
```

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `default-avatar.png`
- `user-john-doe.jpg`
- `placeholder-avatar.svg`
