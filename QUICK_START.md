# Quick Start Guide

Welcome to the Opulent Horizons Image Hosting Repository! This guide will help you get started quickly.

## üöÄ Getting Started in 3 Steps

### Step 1: Choose Your Category

Navigate to the appropriate folder for your image:
- `images/logos/` - Company/product logos
- `images/photos/` - General photographs
- `images/icons/` - UI icons and symbols
- `images/banners/` - Website/email banners
- `images/backgrounds/` - Background images
- `images/avatars/` - User profile pictures

### Step 2: Add Your Image

1. **Optimize** your image first (use TinyPNG, ImageOptim, etc.)
2. **Name** it properly: lowercase, hyphens, descriptive (e.g., `company-logo.png`)
3. **Upload** to the appropriate category folder
4. **Commit** with a clear message (e.g., "Add company logo")

### Step 3: Use Your Image

Access your image via the raw GitHub URL:

```
https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/[category]/[filename]
```

**Example:**
```html
<img src="https://raw.githubusercontent.com/chris1-commits/Opulent-Horizons-Logo-Hosting/main/images/logos/my-logo.png" alt="Logo">
```
